# PawMind v2.0 AI 健康管理 — 设计规格

> 本文件是 brainstorming 阶段产出的设计规格摘要，完整设计详见对应文档。

## 设计决策记录

| 决策项 | 结论 | 理由 |
|--------|------|------|
| v2.0 定位 | AI 健康管理（非社区/行为分析） | 先做个人健康管理，社区功能推迟到 v3.0 |
| 原 AI 陪伴 | 彻底移除「宠物视角对话」 | CareAI 是全新的健康导向模块 |
| AI 能力 | v2.0 Mock 实现 | UI 和流程先到位，AI 能力后续接入 |
| 设备商城 | 展示型，跳外部链接购买 | v2.0 不做交易闭环 |
| 电子疫苗本 | 条形码 OCR 预填 + 手动补充 | 不对接疫苗数据库 |
| 每日小知识 | 固定知识库，预留 AI 生成接口 | v2.0 先跑通内容分发流程 |
| 底部导航 | 「AI聊天」替换为「CareAI」 | 直接替换，其他 Tab 不变 |
| 架构策略 | 渐进增强（非完全重建） | 复用 v1.0 实体和代码，降低风险 |
| 健康指标 | 设备驱动显示 + 营销引导 | 未绑定设备的指标显示解锁引导 |
| 健康档案 | 就诊记录 + 日常观察 | 支持打字/OCR/语音三种输入 |
