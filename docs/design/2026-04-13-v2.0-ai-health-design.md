# PawMind v2.0 AI 健康管理 技术设计

## 目标

在 v1.0 基础上渐进增强，将健康管理从简单记录升级为多渠道数据采集 + 可视化 + AI 辅助诊断平台，同时用 CareAI 模块替换原 AI 陪伴模块。

## 方案概述

**架构策略：渐进增强**

- 保留 v1.0 的 User、Pet、GrowthRecord、HealthLog 实体
- 删除 Conversation、Message 实体（原 AI 陪伴）
- 扩展 health 模块，新增 device、care-ai、vaccination 子模块
- 前端底部导航「AI陪伴」Tab 替换为「CareAI」

**技术栈不变：**
- 前端：React Native + Expo + Zustand
- 后端：NestJS + TypeORM + PostgreSQL
- 共享：@pawmind/shared TypeScript 工具包

#### DeviceProduct（设备商品）

```typescript
@Entity('device_products')
export class DeviceProduct {
  @PrimaryGeneratedColumn()
  id: number;

  @Column({ length: 100 })
  name: string;

  @Column({ length: 30 })
  type: string; // feeder | fountain | collar | litter_box

  @Column({ length: 50 })
  brand: string;

  @Column('text')
  description: string;

  @Column({ type: 'jsonb', nullable: true })
  specs: Record<string, string>; // 产品参数

  @Column({ nullable: true })
  imageUrl: string;

  @Column()
  purchaseUrl: string; // 外部购买链接

  @Column({ type: 'decimal', precision: 10, scale: 2, nullable: true })
  price: number;

  @CreateDateColumn()
  createdAt: Date;
}
```

#### Device（用户绑定设备）

```typescript
@Entity('devices')
export class Device {
  @PrimaryGeneratedColumn()
  id: number;

  @Column()
  userId: number;

  @JoinColumn({ name: 'userId' })
  user: User;

  @Column()
  petId: number;

  @JoinColumn({ name: 'petId' })
  pet: Pet;

  @Column({ nullable: true })
  productId: number;

  @ManyToOne(() => DeviceProduct, { nullable: true })
  @JoinColumn({ name: 'productId' })
  product: DeviceProduct;

  @Column({ length: 50 })
  name: string;

  @Column({ length: 30 })
  deviceType: string; // feeder | fountain | collar | litter_box

  @Column({ length: 100, nullable: true })
  serialNumber: string;

  @Column({ length: 20, default: 'offline' })
  status: string; // online | offline

  @Column({ type: 'int', nullable: true })
  batteryLevel: number; // 0-100

  @Column({ length: 20, nullable: true })
  networkStatus: string; // wifi | bluetooth

  @Column({ type: 'timestamp', nullable: true })
  bindTime: Date;

  @CreateDateColumn()
  createdAt: Date;

  @UpdateDateColumn()
  updatedAt: Date;
}
```

## 关键实现

### AI 服务抽象层

```typescript
/** AI 服务接口，v2.0 用 Mock 实现，后续替换为真实 AI */
interface IAiService {
  /** 健康问答 */
  answerHealthQuestion(
    question: string,
    petContext: PetContext,
  ): Promise<string>;

  /** 图像诊断 */
  analyzeDiagnosisImage(
    imageUrl: string,
    diagnosisType: DiagnosisType,
  ): Promise<DiagnosisResult>;
}
```

### Mock 知识库匹配

- 预置 50-100 条常见宠物健康 Q&A
- 使用关键词匹配（品种、症状、行为等关键词）
- 匹配不到时返回默认引导语

### 设备配对 Mock

- v2.0 配对过程为 Mock 模拟
- 用户选择设备类型后，展示 2-3 步配对动画
- 模拟 3 秒等待后返回配对成功
- 绑定后设备数据由种子数据填充

### 健康指标汇总算法

- 24h：取最近 24 小时内各指标的最新值、最大值、最小值、平均值
- 7天：取最近 7 天每日各指标的平均值，计算趋势（与上一个 7 天对比）
